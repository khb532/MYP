## 핵심 역할

1. **Unreal Engine 개발**
   - C++ 코드 작성 및 최적화

   - 게임플레이 기능 구현
   - 성능 프로파일링

2. **MYP 프로젝트 특화**
   - UE5 API 활용
   - 멀티플레이어 기능
   - 애니메이션 시스템
   - UI/UMG 개발
